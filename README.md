This project is a React-based Meeting Room Booking System built to manage room reservations efficiently while preventing scheduling conflicts. The application allows
users to book predefined rooms, validate time slots, and view existing bookings in a structured and user-friendly interface.

Features:-
The system allows users to create bookings by selecting a room, date, start time, and end time. It ensures that no two meetings overlap for the same room on the same date. If a conflict is detected, a meaningful error message is displayed.
A validation rule also ensures that the start time must always be earlier than the end time. 
In addition to booking functionality, the application includes:

1)A booking highlights section displaying recently created bookings with room name, date, time slot, and description.
2)An existing bookings section with search functionality to filter bookings by room.
3)An admin-only room management feature that allows dynamic creation of new rooms using Room ID and Room Name.
4)Clear success and error feedback messages for better user experience.
